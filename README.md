Credit Trust: A Digital Credit Transparency System
Credit Trust is a robust mobile application designed to bridge the gap between local shopkeepers and their customers. By digitizing traditional "khata" (ledger) books, it provides a transparent, real-time, and secure way to manage credit records.

🚀 Key Features
Real-Time Tracking: Instant updates for both shopkeepers and customers whenever a credit entry is made.

Dual Access: Dedicated interfaces for shop owners to manage records and for customers to view their outstanding balances.

Secure Cloud Storage: Powered by Firebase, ensuring data is never lost even if a device is changed.

Transparency: Reduces disputes by providing a clear history of all transactions and payments.

📦 Installation & Setup
To run this project locally, follow these steps:

Clone the repository:

Bash
git clone https://github.com/shruti-tiwari761/Credit-Trust-A-Digital-Credit-Transparency-System-for-Local-Shops-Flutter-Dart-.git
Navigate to the project folder:

Bash
cd credit_trust
Install dependencies:

Bash
flutter pub get
Configure Firebase:

Create a project on the Firebase Console.

Add an Android/iOS app to the project.

Download the google-services.json (for Android) or GoogleService-Info.plist (for iOS) and place them in the respective directories.

Run the app:

Bash
flutter run
