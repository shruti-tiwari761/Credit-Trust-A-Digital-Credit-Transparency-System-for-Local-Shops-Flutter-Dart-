## 📖 Project Overview
**Credit Trust** is a modern mobile solution designed to replace traditional paper-based "Khata" (ledger) books. Developed using **Flutter & Dart**, this application provides a transparent, real-time, and reliable platform for managing credit records.

---

## 🚀 Key Features
- ⚡ Real-Time Tracking
- 👥 Dual Access System
- 🔒 Cloud Storage (Firebase)
- 🛡️ User Authentication
- 📊 Easy Record Management

---

## 🛠️ Tech Stack
- Flutter (Dart)
- Firebase
- Cloud Firestore
- Firebase Auth

## 📦 Installation & Setup

### 1️⃣ Clone the Repository
    git clone https://github.com/shruti-tiwari761/Credit-Trust-A-Digital-Credit-Transparency-System-for-Local-Shops-Flutter-Dart-.git

### 2️⃣ Navigate to Project Folder
    cd Credit-Trust-A-Digital-Credit-Transparency-System-for-Local-Shops-Flutter-Dart-

### 3️⃣ Install Dependencies
    flutter pub get

### 4️⃣ Firebase Configuration
- Add google-services.json in:
    android/app/
- Add GoogleService-Info.plist in:
    ios/Runner/

### 5️⃣ Run the App
    flutter run
